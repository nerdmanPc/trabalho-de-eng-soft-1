USUARIO
  - MEMBROS:
    - CODIGO
    - NOME
  - ALUNO GRADUACAO: 
    - 1 DIA DE EMPRESTIMO
    - 3 LIVROS MAX
  - ALUNO POS: 
    - 2 DIAS DE EMPRESTIMO
    - 4 LIVROS MAX
  - PROFESSOR:
    - 4 DIAS DE EMPRESTIMO
    - SEM LIMITE DE LIVROS

LIVRO
  - MEMBROS:
    - CODIGO 
    - TITULO
    - EDITORA
    - AUTORES
    - EDICAO
    - ANO PUBLIC

EXEMPLAR

RESERVA

EMPRESTIMO:
  - MEMBROS:
    - PRAZO DEVOLUCAO

FUNCIONALIDADE EMPRESTIMO:
  - COMANDO emp [usu] [lvr]
  - SE USUARIO FEZ RESERVA, REMOVE
  - CRIA EMPRESTIMO
  - IMPRIME SE FUNCIONOU COM NOME E TITULO
  - SE NAO FUNCIONA, DIZ O MOTIVO
  - CRITERIOS PARA ALUNO:
    - EXEMPLAR DISPONIVEL
    - USUARIO NAO JUBILADO
    - USUARIO PODE PEGAR MAIS LIVROS
    - SE USUARIO NAO TEM RESERVA, HÁ LIVROS NAO
      RESERVADOS OU USUARIO TEM RESERVA
    - USUARIO NAO ESTA COM EXEMPLAR DO LIVRO
  - CRITERIOS PARA PROFESSOR:
    - EXEMPLAR DISPONIVEL
    - PROFESSOR NAO JUBILADO

FUNCIONALIDADE DEVOLUCAO:
  - COMANDO dev [usu] [lvr]
  - IMPRIME SE FUNCIONOU COM NOME E TITULO
  - SE NAO FUNCIONA, DIZ O MOTIVO
  - SO FALHA SE NAO HOUVER EMPRESTIMO DO [lvr] POR [usu]

FUNCIONALIDADE RESERVA:
  - COMANDO res [usu] [lvr]
  - ATE 3 LIVROS POR USUARIO
  - IMPRIME SE FUNCIONOU
  - SE NAO FUNCIONA, DIZ O MOTIVO

FUNCIONALIDADE SUBSCRIBE:
  - COMANDO obs [usu] [lvr]
  - PROFESSORES PODEM SER OBSERVADORES DE LIVROS
  - OBSERVADORES SÃO NOTIFICADOS TODA VEZ QUE O LIVRO TEM MAIS DE DUAS RESERVAS SIMULTANEAS
  - OBSERVADORES APENAS GUARDAM QUANTAS VEZES FORAM NOTIFICADOS

FUNCIONALIDADE CONSULTA LIVRO:
  - COMANDO liv [lvr]
  - IMPRIME O SEGUINTE:
    - TITULO DO LIVRO
    - NUMERO DE RESERVAS E NOME DE USUARIOS
    - PARA CADA EXEMPLAR:
      - CODIGO
      - STATUS DE EMPRESTADO
      - USUARIO QUE FEZ O EMPRESTIMO
      - DATA DE EMPRESTIMO
      - DATA DE DEOLUCAO\

FUNCIONALIDADE CONSULTA USUARIO:
  - COMANDO usu [usu]
  - PARA CADA EMPRESTIMO:
    - TITULO DO LIVRO
    - DATA DE EMPRESTIMO
    - DATA DEVOLUCAO
  - PARA CADA RESERVA:
    - TITULO DO LIVRO
    - DATA DE SOLICITACAO

FUNCIONALIDADE CONSULTA PROFESSOR:
  - COMANDO ntf [usu]
  - IMPRIME QUANTIDADE DDE NOTIFICACOES
